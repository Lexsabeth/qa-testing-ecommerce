# Bug Report

ID: BUG-LOGIN-001

Título  
El botón de login no responde al hacer clic.

Ambiente

Navegador: Chrome  
Sistema operativo: Windows 10  

---

Pasos para reproducir

1. Abrir la página de login
2. Ingresar usuario y contraseña
3. Presionar botón "Ingresar"

---

Resultado esperado

El sistema debería permitir el acceso al usuario.

---

Resultado actual

El botón no ejecuta ninguna acción.

---

Severidad

Alta
