# Estrategia de Testing - Ecommerce

## Objetivo

Validar el correcto funcionamiento de un sistema de ecommerce asegurando que los usuarios puedan navegar, buscar productos y realizar compras sin errores.

## Alcance

Las pruebas cubrirán:

- Registro de usuario
- Inicio de sesión
- Búsqueda de productos
- Carrito de compras
- Proceso de checkout

## Tipos de pruebas

Pruebas funcionales  
Pruebas exploratorias  
Pruebas de regresión   

## Herramientas utilizadas

Jira  
Postman  
SQL
